# Sistem POS Kafe Terpadu (Full-Stack Kotlin)

## Masalah & Solusi
- **Real-time Monitoring**: Menggunakan WebSockets/Firebase agar data terpantau langsung.
- **Security**: Implementasi JWT untuk otorisasi multi-user dan multi-device.
- **Transparency**: Integrasi payment gateway untuk validasi transaksi otomatis.

## Teknologi
- **Backend**: Kotlin Spring Boot
- **Frontend**: Android Kotlin Jetpack Compose
- **Database**: mysql
